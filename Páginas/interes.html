    • Red informática

    Una red informática es un conjunto de dispositivos conectados entre sí con el objetivo de compartir información, recursos y servicios, como archivos, impresoras o conexión a Internet.

    Tipos de redes: LAN y WAN. ¿Qué es Internet?

    LAN (Local Area Network): Red que conecta dispositivos en un espacio reducido, como una casa, un instituto o una oficina.

    WAN (Wide Area Network): Red que conecta varias redes LAN a grandes distancias, incluso entre países.

    Internet: Es una red WAN mundial que conecta millones de dispositivos y redes en todo el planeta.

    Proveedor de servicios de Internet (ISP)

    Un proveedor de servicios de Internet es la empresa que proporciona conexión a Internet a los usuarios.

    Ejemplos: Movistar, Vodafone, Orange, Digi, Jazztel.

    Dominio en Internet

    Un dominio es el nombre que identifica a una página web en Internet y permite acceder a ella de forma sencilla.

    Ejemplos: google.com, youtube.com

    Protocolo informático

    Un protocolo informático es un conjunto de normas que establecen cómo se envía y recibe la información entre dispositivos conectados a una red.

    Dirección IP

    Una dirección IP es un número que identifica a un dispositivo dentro de una red.

    IP pública: Identifica un dispositivo en Internet y es visible desde el exterior.

    IP privada: Se utiliza dentro de una red local y no es accesible desde Internet.

    URL

    Una URL es la dirección completa que indica la ubicación de un recurso en Internet y el método para acceder a él.

    Ejemplo: https://www.wikipedia.org

    Protocolo DNS

    El protocolo DNS se encarga de traducir los nombres de dominio en direcciones IP para que los dispositivos puedan localizar los servidores web.

    Protocolo HTTP y HTTPS

    HTTP: Permite la comunicación entre el navegador y el servidor web.

    HTTPS: Versión segura de HTTP que cifra la información para proteger los datos del usuario.

    Página web estática vs página web dinámica

    Página web estática: Muestra siempre el mismo contenido y no cambia.

    Página web dinámica: El contenido cambia según el usuario, la interacción o una base de datos.

    Alojamiento web / Hosting

    El alojamiento web es el servicio que permite guardar una página web en un servidor para que esté disponible en Internet.

    Ejemplos: Hostinger, OVH, SiteGround, Bluehost.

    CMS

    Un CMS es un sistema que permite crear y gestionar páginas web de forma sencilla sin programar desde cero.

    Ejemplos: WordPress, Joomla, Drupal.

    Ventajas:

    • Fácil de usar
    • Ahorra tiempo
    • No requiere conocimientos avanzados

    Inconvenientes:

    • Menor personalización
    • Puede ser más lento
    • Dependencia de plugins y actualizaciones
